Zencoder .NET

A .NET C# client library for the zencoder.com API

Zencoder .NET is a fully object-oriented implementation of the Zencoder API for .NET. It is compatible with .NET 3.5 and above, has both synchronous and asynchronous APIs, and includes support for receiving and parsing HTTP notifications.
