Serialization.Plists

Binary plist serialization and de-serialization for C# and .NET

Serialization.Plists is a binary plist read/writer implementation for .NET. It supports the complete plist specification that Apple has published, with two caveats:

  • Serialization of opaque data or non plist-compatible objects will break plist editing in Property List Editor
  • Sets are treated as arrays
