Jacket

Release your inner album.

Jacket helps bands release music in the post-CD apocalypse. It is an app and a server application that work together to enable independent musicians to release their albums directly to fans at shows. Albums are delivered to fans' smartphones, with zero intervention, when they are in the audience of a Jacket-powered show.

Jacket is being built with MonoTouch and Windows Azure.
